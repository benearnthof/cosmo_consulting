inla.dBind {INLA}R Documentation

Build a block-diagonal sparse matrix.

Description

Build a block-diagonal sparse matrix. Obsolete wrapper for bdiag().

Usage

inla.dBind(...)

Arguments

...

A list of square or rectangular matrices.

Value

A sparse matrix.

Author(s)

Finn Lindgren finn.lindgren@gmail.com

Examples


inla.dBind(Matrix(1, 2, 1), Matrix(2, 1, 2))
